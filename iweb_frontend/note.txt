1.面试题：大型页面时的章法？
  一个大型的页面 （列如：京东首页） css 大约有1000行，编写时有一定的章法：
   ①reset： 样式重置，把html标签的默认样式统一化——可以使用Normalize.css
   ②scaffoding：样式脚手架，设定全局的基础样式

2.使用浮动布局时常见的问题
若容器中的子元素浮动起来，则不再占用布局空间；
如果容器内所有的子元素全部浮动，则父容器不再有高度。
.child{
	flat:left;
}
.parent:after {/*在父容器尾部生成新孩子*/
	content:'';
	display:block;
	clear:both;/*清楚兄弟元素浮动造成的影响*/
}
